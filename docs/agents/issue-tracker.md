# Issue tracker

**Type**: GitHub Issues
**Repo**: `rajput2122/SocialMediaSearch`
**CLI**: `gh` (GitHub CLI)

## Usage

```bash
# Create an issue
gh issue create --title "..." --body "..." --label "needs-triage"

# List open issues
gh issue list

# Edit labels on an issue
gh issue edit <number> --add-label "ready-for-agent" --remove-label "needs-triage"

# View an issue
gh issue view <number>
```
